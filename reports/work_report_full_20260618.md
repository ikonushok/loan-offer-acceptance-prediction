# Отчёт о проделанной работе — Alfa Bank x MFTI «Отклик на кредитный оффер»

**Дата:** 2026-06-18
**Метрика:** ROC-AUC | **Лучший публичный результат:** 0.76054 | **Ориентир:** ~0.775

---

## 1. Задача

Предсказать вероятность принятия корпоративным клиентом кредитного оффера и ранжировать офферы по этой вероятности. Сабмит — только по строкам `test_apps.csv` (36 311), подтверждено организаторами; `sample_submission.csv` (45 032) — лишь ориентир формата.

---

## 2. Характеристики данных (что выяснено)

- **train:** 145 241 строка, 28 колонок; **test:** 36 311 строк, 27 колонок. Доля позитивов в train = 6.1%.
- **Данные предобработаны организаторами**: признаки центрированы ~0, ограниченные диапазоны, низкий skew → нормализация и обрезка выбросов уже сделаны провайдером. Дополнительная нормализация для деревьев не нужна.
- **Пропуски значительны**: от 20% до 96% NaN в ряде колонок (overdraft_app_term_max_360 — 96%, loan_rev_* — 86-91%). GBM используют NaN нативно (числовые не импутируются).
- **Групповая структура** (один запрос → N офферов) реальна, но мала: multi-offer группы = 6.1% строк; acceptance внутри них НЕ «выбор 1 из N» (никогда >1 принятого, доля даже ниже). train/test пересечение групп = 0.
- **Сильный временной дрейф** — главная особенность задачи:
  - train заканчивается 2025-06-05, test идёт до 2025-12-01 (почти весь test — будущее);
  - adversarial AUC train-vs-test = 0.74 (без даты), `cb_rate` adversarial AUC = 0.63;
  - prevalence нестационарна 5×: 0.031 (фев'24) → 0.152 (дек'24) → 0.07-0.09 (апр-июн'25);
  - связь фича→таргет распадается во времени (`loan_rev_max_start_non_fin`: corr 0.34 → 0.00 к Fold3).

---

## 3. Методология валидации

- **Random CV ненадёжен** из-за временного дрейфа → отвергнут для отбора моделей.
- **Rolling time folds** (cutoffs 2025-01-01, 2025-03-01, 2025-04-01): Fold3 (2025-04-01..06-05) — самый «будущеподобный», главный критерий отбора. OOF — вторичный.
- **Late holdouts** H1/H2/H3 (растущие поздние периоды) — дополнительная будущеориентированная проверка.
- **Адверсариальный аудит дрейфа**: подтверждены нулевое пересечение групп train/test и дрейф offer-экономики.
- Деградация по фолдам: Fold1 0.796 → Fold2 0.791 → Fold3 0.752 (реальный концепт-дрейф).

---

## 4. Моделирование

### Champion
Бленд CatBoost (без raw time-фич, context-offer features, Fold3-ориентированный HPO):
`0.70 × trial0070 + 0.04 × trial0041 + 0.26 × trial0164` (raw probability blend).
- Fold1/2/3 = 0.796 / 0.793 / 0.755, OOF = 0.7811.
- Конфиг: context-offer features включены, pairwise features и missing-flags выключены, decision_day исключён.

### HPO
- CatBoost Optuna 200 trials + расширение ext1 (150 trials, seed=137) — потолок Fold3 ≈ 0.7536, не превзошёл champion.
- LGBM Optuna 100 trials — Fold3 0.7456 → 0.7520 (слабее CatBoost/XGB).
- XGB HPO — лучший depth3_child80_reg30, Fold3 standalone 0.7550.

### Диверсификация для бленда
- XGB, LGBM, ExtraTrees — все коррелируют >0.87 с champion (XGB/LGBM между собой 0.976). Бленд-прирост ограничен +0.002 Fold3.
- Лучший бленд-кандидат: **xgb_rank_c62_x38** (champion_rank×0.62 + xgb_unweighted_rank×0.38): Fold3 0.7560, lh_mean 0.7631 (+0.0029 над champion), red-team пройден.

## 7. Ключевой вывод

**Достигнут потолок текущих данных.** Решающее доказательство: **test-weighted AUC champion = 0.7755 ≈ ориентир 0.775**. На тест-подобных данных модель уже ранжирует хорошо — разрыв до публичного 0.76054 это **необратимая сложность периода Q4 2025** (концепт-дрейф), которую нельзя закрыть пересборкой имеющихся данных. Дополнительных данных не будет.

Все попытки адаптации к дрейфу (recency, importance weighting, pseudo-labeling, дроп дрейф-фич) не дали прироста — модель уже извлекает устойчивый сигнал близко к пределу. Tabular NN не дал полезной декорреляции (слаб). Больше фолдов улучшит измерение/отбор, но не научит модель ловить будущий сдвиг.
